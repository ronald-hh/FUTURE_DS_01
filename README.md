# Online Retail Sales Analysis

## Project Overview

This project analyzes an online retail sales dataset using Python. It cleans the data, explores sales trends, and finds useful business insights through charts and summary statistics.

## Features

- Clean and prepare the dataset
- Calculate important sales statistics
- Analyze revenue over time
- Find the top-selling products
- Identify the countries with the highest sales
- Find the top customers
- Provide business insights and recommendations

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib

## Installation

Install the required libraries before running the notebook.

```bash
pip install pandas numpy matplotlib notebook
```

## How to Run

1. Download or clone this repository.

```bash
git clone <repository-link>
```

2. Move into the project folder.

```bash
cd <project-folder>
```

3. Start Jupyter Notebook.

```bash
jupyter notebook
```

4. Open the file:

```
Online_Retail_Analysis.ipynb
```

5. Run the notebook from top to bottom.

## Dataset

The project uses an online retail sales dataset containing information such as:

- Invoice number
- Product description
- Quantity sold
- Unit price
- Customer ID
- Country
- Invoice date

This project uses the **Online Retail Dataset**, which contains transactions from a UK-based online retail store.
**Source:**  
https://www.kaggle.com/datasets/ulrikthygepedersen/online-retail-dataset

## Project Structure

```
Online_Retail_Analysis.ipynb
README.md
dataset.csv
```

> Replace `dataset.csv` with the actual dataset filename if it is different.
